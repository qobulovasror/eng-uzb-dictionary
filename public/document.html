English-Uzbek Dictionary Documentation

API Endpoints

/api/words

Returns a list of all available words and their translations in the English-Uzbek dictionary.

Method: GET

Response

An array of objects, each containing:

  • eng: The English word
  • uzb: The corresponding translation in Uzbek

/api/words/:name

Returns the translation of the specified English word in Uzbek.

Method: GET

Parameters

  • :name (string, required): The English word to search for

Response

An object containing:

  • eng: The English word
  • uzb: The corresponding translation in Uzbek

Usage Example

Search a Word

Endpoint: /api/words/:name

Example

GET /api/words/hello
Response:
{
  "eng": "hello",
  "uzb": "salom"
}
